  # 任务：生成类网易云音乐平台前端页面                                                           
                                                                                                 
                                                                                                 
                                                                                                 
  你是一个资深前端工程师，精通 Vue 3 生态。请使用 Vue 3 + Composition API + Vite                 
                                                                                                 
  构建一个类网易云音乐的Web音乐平台前端。                                                        
                                                                                                 
                                                                                                 
                                                                                                 
  ## 技术栈                                                                                      
                                                                                                 
                                                                                                 
  - Vue 3（Composition API + `<script setup>`）                                                  
                                                                                                 
  - Vite 构建工具                                                                                
  - Vue Router 4（路由管理）                                                                     
                                                                                                 
  - Pinia（状态管理）                                                                            
                                                                                                 
  - Axios（HTTP 请求）                                                                           
                                                                                                 
  - CSS 预处理器：SCSS                                                                           
                                                                                                 
  - 组件库：Element Plus（用于表单、弹窗等通用 UI 组件）                                         
                                                                                                 
  - 图标库：使用 SVG 图标或 iconfont                                                             
                                                                                                 
                                                                                                 
                                                                                                 
  ## 参考页面                                                                                    
                                                                                                 
                                                                                                 
                                                                                                 
  以 https://music.163.com/#/discover 为设计参考，**1:1 复刻**首页的 UI 布局和视觉风格：         
                                                                                                 
  - 顶部导航栏（logo、导航菜单、搜索框、用户头像/登录入口）                                      
  - 左侧/主内容区轮播图 Banner                                                                   
                                                                                                 
  - 推荐歌单卡片网格                                                                             
                                                                                                 
  - 最新音乐列表                                                                                 
                                                                                                 
  - 底部固定音乐播放控件（进度条、播放/暂停、上下曲、音量控制、播放列表）                        
                                                                                                 
  - 整体配色、字体、间距、圆角、阴影等细节尽量还原                                               
                                                                                                 
                                                                                                 
                                                                                                 
  ## 需要实现的页面                                                                              
                                                                                                 
                                                                                                 
                                                                                                 
  ### 1. 首页（/discover）                                                                       
                                                                                                 
  - 完整复刻网易云音乐发现页 UI                                                                  
  - 轮播 Banner（可使用 Element Plus 的 Carousel 或手写）                                        
                                                                                                 
  - 推荐歌单模块（卡片网格布局）                                                                 
                                                                                                 
  - 最新音乐/新碟上架模块                                                                        
                                                                                                 
  - 所有数据可先用 Mock 数据填充，预留 API 接口调用位置                                          
                                                                                                 
                                                                                                 
                                                                                                 
  ### 2. 底部音乐播放控件（全局组件）                                                            
                                                                                                 
  - 固定在页面底部，所有路由页面均可见                                                           
                                                                                                 
  - 显示：专辑封面缩略图、歌曲名、歌手名                                                         
                                                                                                 
  - 控制按钮：上一曲、播放/暂停、下一曲                                                          
                                                                                                 
  - 进度条（可拖拽跳转）                                                                         
                                                                                                 
  - 音量控制滑块                                                                                 
                                                                                                 
  - 播放模式切换（列表循环/随机/单曲循环）                                                       
                                                                                                 
  - 播放列表展开按钮                                                                             
                                                                                                 
  - 歌词展示区域（可选）                                                                         
                                                                                                 
                                                                                                 
                                                                                                 
  ### 3. 用户登录页面（/login）                                                                  
                                                                                                 
  - 支持用户名/邮箱 + 密码登录                                                                   
                                                                                                 
  - 表单验证（必填项、邮箱格式校验）                                                             
                                                                                                 
  - 登录错误提示（用户不存在、密码错误等）                                                       
                                                                                                 
  - "没有账号？去注册"跳转链接                                                                   
                                                                                                 
  - 登录成功后跳转到首页或之前的页面                                                             
                                                                                                 
                                                                                                 
                                                                                                 
  ### 4. 用户注册页面（/register）                                                               
                                                                                                 
  - 邮箱 + 验证码 + 密码 + 确认密码                                                              
                                                                                                 
  - 「发送验证码」按钮（含 60 秒倒计时）                                                         
                                                                                                 
  - 密码强度校验（至少 6 位，含字母和数字）                                                      
                                                                                                 
  - 两次密码一致性校验                                                                           
                                                                                                 
  - 注册成功后跳转到登录页                                                                       
                                                                                                 
                                                                                                 
                                                                                                 
  ### 5. 个人中心页面（/user）                                                                   
                                                                                                 
  - 显示用户头像、昵称、邮箱等基本信息                                                           
                                                                                                 
  - 修改密码功能（旧密码 + 新密码 + 确认新密码）                                                 
                                                                                                 
  - 简单的信息展示卡片布局                                                                       
                                                                                                 
                                                                                                 
                                                                                                 
  ## 数据 Mock 策略                                                                              
                                                                                                 
                                                                                                 
                                                                                                 
  在 API 接口未就绪之前，所有页面数据通过 Mock 方式提供：                                        
                                                                                                 
  - 在 `src/api/` 目录下封装所有 API 请求函数                                                    
  - 使用拦截器或环境变量切换 Mock / 真实 API                                                     
                                                                                                 
  - Mock 数据结构参考网易云音乐 API 的实际返回格式                                               
                                                                                                 
  - 歌单、歌曲、Banner 等数据至少准备 6-10 条示例数据                                            
                                                                                                 
                                                                                                 
                                                                                                 
  ## 代码组织要求                                                                                
                                                                                                 
                                                                                                 
                                                                                                 
  src/                                                                                           
                                                                                                 
  ├── api/          # API 请求封装（含 mock 数据）                                               
  ├── assets/       # 静态资源（图片、图标、全局样式）                                           
                                                                                                 
  ├── components/   # 公共组件（Player、Navbar、SongCard 等）                                    
                                                                                                 
  ├── composables/  # 组合式函数（播放器逻辑、用户状态等）                                       
                                                                                                 
  ├── router/       # 路由配置                                                                   
                                                                                                 
  ├── stores/       # Pinia 状态管理                                                             
                                                                                                 
  ├── views/        # 页面级组件                                                                 
                                                                                                 
  ├── App.vue                                                                                    
                                                                                                 
  └── main.js                                                                                    
                                                                                                 
                                                                                                 
                                                                                                 
  ## 样式要求                                                                                    
                                                                                                 
                                                                                                 
                                                                                                 
  - 使用 SCSS 编写样式                                                                           
                                                                                                 
  - 全局颜色、字体、间距定义为 SCSS 变量                                                         
                                                                                                 
  - 响应式布局：适配 1280px - 1920px 主流分辨率                                                  
                                                                                                 
  - 组件级 scoped 样式，避免污染                                                                 
                                                                                                 
  - 网易云标志性的红色主题色（#C62F2F 或近似值）                                                 
                                                                                                 
  - 暗色/亮色均可，默认采用网易云的白色+红色主题                                                 
                                                                                                 
                                                                                                 
                                                                                                 
  ## 交互细节                                                                                    
                                                                                                 
                                                                                                 
                                                                                                 
  - 页面切换使用 router-view，保持播放控件始终可见                                               
                                                                                                 
  - 登录/未登录状态在导航栏有明显区分                                                            
  - 播放控件在无歌曲播放时显示默认状态                                                           
                                                                                                 
  - 表单提交时按钮显示 loading 状态防重复提交                                                    
                                                                                                 
  - 所有可点击区域有 hover 效果                                                                  
                                                                                                 
                                                                                                 
                                                                                                 
  ## 输出要求                                                                                    
                                                                                                 
                                                                                                 
                                                                                                 
  请按以下顺序逐步完成：                                                                         
                                                                                                 
  1. 先搭建项目脚手架（Vite + Vue 3）                                                            
  2. 配置路由、状态管理、全局样式变量                                                            
                                                                                                 
  3. 实现底部全局播放控件组件                                                                    
                                                                                                 
  4. 实现首页发现页面（discover）                                                                
                                                                                                 
  5. 实现登录页面                                                                                
                                                                                                 
  6. 实现注册页面                                                                                
                                                                                                 
  7. 实现个人中心页面                                                                            
                                                                                                 
  8. 添加 Mock 数据并确保所有页面数据流通
